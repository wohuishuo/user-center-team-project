# 技术栈图鉴

每个技术一张"卡片",用**统一的五段式**讲清楚,让你看懂技术之间的**依赖关系网**,而不是孤立地记名词。

## 五段式结构

每张技术卡都回答这五个问题:

1. **是什么** —— 一句话定义
2. **解决什么问题** —— 没有它会怎样
3. **依赖谁 / 被谁依赖** —— 在技术栈里的位置
4. **版本与生命周期** —— 2026 该用哪个版本
5. **在用户中心里的角色** —— 我们项目具体怎么用

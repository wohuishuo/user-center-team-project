## 文档写作要点(来自示例文档反馈)

::: warning 容易被扣分的点
- **数据安全设计**:针对不同访问者/访问类型/数据对象,分别说明查看、获取、传输时的安全措施(保密、授权、防越权)。
- **文字 + 图表结合**:不要只堆图表,要对图表含义、适用范围做文字说明。
- **数据标准化**:领域专业术语是否有统一标准(如医学术语、聊天术语如何规范)。
- **图表必须有解释**:技术文档不是图的合集,每张图都要讲清"它说明了什么"。
:::
